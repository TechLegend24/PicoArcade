This is the code for a game console with the Pico.
I have three buttons connected to ground and GP16, GP20, and GP26
I have an I2C SSD1306 OLED screen on GP18 and GP19
I have supplied some libraries, but you need adafruit_register, adafruit_bus, and adafruit_display_text
This code was written in CircuiPython 10 for the Raspberry Pi Pico 1
